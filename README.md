            <br>

<h2 align="center">Steps<h2>

<h3>The Console</h3>

1. Create data model.
2. Manage (Create, update, destroy, etc).
3. Serialize and store objects to ```JSON``` file. 

Behind this, the idea is create a CLI to manage and store data.

<hr>
<br>

<h2>Files and Directories</h2>

1. ```Models``` directory will contain all classes used
2. ```Tests``` directory will contain all unit tests.
3. ```console.py``` file is the entry point of our command interpreter.
4. ```models/base_model.py``` this file is our base model class. And contains:
    1.1 attributes: ```id```, ```created_at``` and ```updated_at```
    1.2 methods:  as ```save()``` and ```to_json()```
    
    
    
<h2>How to Execute</h2>

Your shell should work like this in interactive mode:

```
$ ./console.py
(hbnb) help

Documented commands (type help <topic>):
========================================
EOF  help  quit

(hbnb) 
(hbnb) 
(hbnb) quit
$
```

But also in non-interactive mode:

```
$ echo "help" | ./console.py
(hbnb)

Documented commands (type help <topic>):
========================================
EOF  help  quit
(hbnb) 
$
$ cat test_help
help
$
$ cat test_help | ./console.py
(hbnb)

Documented commands (type help <topic>):
========================================
EOF  help  quit
(hbnb) 
$
```
